# PIANO

A simple web-based piano application that allows you to play musical notes in your browser.

## Requirements

This application requires a live server to run properly. You can use:
- Visual Studio Code with Live Server extension
- Any other HTTP server

## How to Run

1. Clone this repository
2. Open the project folder in Visual Studio Code
3. Install the Live Server extension if you haven't already
4. Right-click on `index.html` and select "Open with Live Server"
5. The application will open in your browser

## Features

- Interactive piano keys
- Play notes by clicking buttons
- Responsive design for mobile and desktop

## Technologies Used

- HTML5
- CSS3
- JavaScript
